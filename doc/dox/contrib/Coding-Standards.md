# Coding Standards {#Coding-Standards}

## Code Organization

### Logical organization

TTG namespaces:

- `ttg`
- `ttg_RUNTIME`, e.g., `ttg_madness` and `ttg_parsec`

### Physical organization

Directory structure:

- `ttg`:
  - `ttg/ttg`:
  - `ttg/ttg/base`:
  - `ttg/ttg/RUNTIME`:
  - `ttg/ttg/common`:
  - `ttg/ttg/util`:
- `tests`:
- `examples`:
